4-Layer Security Pipeline

Every skill goes through all four checks. No exceptions.

1

Static Analysis

AST scan for eval, exec, network calls to unknown hosts, obfuscated code, base64 payloads.

2

Sandboxed Execution

Skill runs in isolated Docker container with network disabled. Monitors system calls and file access.

3

Dependency Audit

All npm/pip dependencies scanned against CVE database. Typosquatting detection on package names.

4

Community Review

3 trusted community members review code before publish. Human eyes catch what scanners miss.
